## What You Get

<CardGrid stagger>
  <Card title="Embedded AI Agent" icon="rocket">
    Claude Code as the brain, Telegram as the dashboard. The agent reads your thesis files, executes tools, manages positions, and learns from each trade. Free-text chat routes to AI; slash commands hit the API directly at zero cost.
  </Card>
  <Card title="42 Daemon Iterators" icon="setting">
    A tick-based engine running account collection, risk gates, SL/TP enforcement, conviction sizing, funding tracking, liquidation monitoring, bot-pattern detection, news ingestion, profit lock, and more — all on a configurable clock.
  </Card>
  <Card title="Conviction Engine" icon="star">
    Position sizing scales with thesis strength (0.0–1.0). Stale theses auto-clamp leverage. Thesis files in `data/thesis/` are the shared contract between you and the system. Kill switch built in.
  </Card>
  <Card title="Mandatory SL & TP" icon="warning">
    Every position MUST have both a stop loss and take profit on the exchange. No exceptions. The daemon enforces this every tick using ATR-based stops and thesis-derived targets.
  </Card>
  <Card title="Hardened Key Management" icon="approve-check">
    Open Wallet Standard vault (AES-256-GCM) + macOS Keychain dual-write. Session tokens only — never API keys. API wallets cannot withdraw funds. If a key leaks, revoke it instantly.
  </Card>
  <Card title="Web Dashboard" icon="laptop">
    Local-only Next.js dashboard on port 3000 with FastAPI backend on port 8420. Real-time charts, log streaming via SSE, iterator control panel, and thesis management — all bound to 127.0.0.1.
  </Card>
</CardGrid>
